Features

Source powered workflow allows developers to code new features directly in the documentation. Source combines front-end interface building with documentation process, making it a powerful tool for productive development and teamwork.

  • Essential Technology Pack:
    NodeJS, Grunt, RequireJS, LESS
  • Convenient navigation with fast document search
  • Feature loaded documentation pages for comfortable and effective team work
  • Interactive style guides
    for designers

How it works

  • Manage your source files just like you would in any modern web project, separating view blocks in different folders with all necessary technologies (html, css, js).
  • Work with familiar environment in .html files and Source will do the rest with NodeJS and client JavaScript.
  • Get feature loaded documentation pages and tools directly in your favorite browser for comfortable team work.
